SGM Lunch Shortcut

Get daily lunch info from SGM using an Apple Shortcut.

Get the Shortcut

iPhone only (Apple Shortcuts).

Before You Start

  • Settings → Shortcuts → turn on “Allow Untrusted Shortcuts” (if you don’t see it, run any Gallery shortcut once, then come back).
  • Settings → Siri & Search → turn on “Listen for ‘Siri’” (or “Hey Siri”).
  • Settings → Siri & Search → Siri Responses → select “Prefer Spoken Responses.”

Install the Shortcut

  • Open the link above on your iPhone.
  • Tap “Get Shortcut” and allow any permissions it asks for.
  • Say “What’s lunch today?” or ask for a specific date.

Data Source

https://joseph-m-valdez.github.io/sgm-lunch/data/YYYY-MM.json

Apple references: Change Siri settings and Intro to Shortcuts.
